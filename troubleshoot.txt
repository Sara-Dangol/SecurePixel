
## 🔧 Troubleshooting

### GUI doesn't appear?
- Make sure VcXsrv is running
- Windows Firewall might block it - allow VcXsrv
- Run `run.bat` again

### Docker errors?
- Run PowerShell as Administrator
- Run: `wsl --update`
- Restart Docker Desktop

### Database connection issues?
- Wait 30 seconds for MySQL to fully start
- Check if ports 3307 and 8080 are free
- Run: `docker-compose logs mysql`
